🚀 Nexian AI Platform

AI-powered platform to analyze skills, take personalized skill tests, and explore earning opportunities based on your skill level.

✨ Features:

🏠 Landing Page: Clean, scrollable UI showcasing platform features.

🎁 Claim Free Skill: Users start for free by providing Name, Email, Phone, and reason to start earning.

🛠 Skill Selection: Choose or type your skill for a personalized experience.

📝 Skill Test:
5 skill-specific questions
Radio-button answers with internal scoring

Determines skill level: Beginner 📚 / Intermediate 👍 / Advanced 🚀
📊 Result & Recommendations: Displays score, level, and personalized advice based on performance.

💰 Opportunities & Earnings:
Beginner: Small tasks (₹50 – ₹500)
Intermediate: Freelancing projects (₹500 – ₹3000)
Advanced: Full client projects (₹3000+)
Special: First ₹1000 Plan included

🎯 Start Exploring Button: Entry point to explore tasks and real earning opportunities.

🛠 Tech Stack

Frontend: React, Framer Motion, Chart.js
Backend: Node.js (for AI queries and skill analysis API)
Deployment: Vercel
